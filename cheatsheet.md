#Git Cheatsheet

### Start a new project

```shell
$ mkdir project_name
$ git init
$ touch readme.md
$ git add readme.md
git commit -m "Initial Commit"
```

### Do some work and then save it

First do some work! 
Make some changes, put them in the box, label the box.
```shell
$ git status
$ git add <whatever file>
$ git status
$ git commit -m "I made a bunch of changes, there are so many details, wee."
```
